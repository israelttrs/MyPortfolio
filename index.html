My Mission

My mission is to master the art of coding, harnessing these skills to create groundbreaking innovations that push the boundaries of what’s possible. I believe technology is the key to shaping the future, and it’s up to us to drive that change. Just as learning to drive empowers you to control a car, learning to code empowers you to shape the digital world. You don’t need to understand every detail of how an engine works to drive; similarly, you don’t need to know every intricate detail of programming to create something incredible. The goal is simple: keep moving forward, avoid crashes, and take control of your journey. Don’t let others steer your course when you have the power to do it yourself.
Skills

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

Cascading Style Sheet or CSS is used to set the style in web pages. It sets the background color, font-size, font-family, color, and other property of elements.

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development.

JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive.
